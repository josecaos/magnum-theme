Pix-Base

Broadcast / Video Devices

Base de montaje ajustable para los monitores de campo de grabación PIX-E5, PIX-E5H y PIX-E7, con o sin el PIX-LR conectado. También se puede usar como soporte para otros dispositivos, como los grabadores de video portátiles PIX 240i y PIX 220i.
  • Diseño de enclavamiento para unir múltiples bases juntas, una al lado de la otra.
  • Nivel burbuja
  • 7 pulgadas de largo, correa de gancho y lazo para asegurar los cables
  • Alturas y ángulos ajustables; Accesible para una fácil portabilidad.
  • Tornillo universal de 1/4 pulgada y 20 hilos con rueda para montar monitores u otros dispositivos similares
  • Agujeros en la base para el montaje a nivel de superficie o pared
  • Incluye una llave hexagonal para hacer ajustes.
  • Peso máximo: 4 lbs (1.8 kg) en extensión completa

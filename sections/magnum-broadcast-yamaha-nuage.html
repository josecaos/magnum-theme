NUAGE

Broadcast / Yamaha

Superficie de control modular para post-producción.
  • Estaciones de audio digital, controladores e interfaces: las funciones profesionales de postproduccion sólo pueden apreciarse cuando se utilizan todos los componentes a la vez y en perfecta armonía. El sistema de producción de audio integrado Nuage reúne el conjunto de requisitos que define los nuevos estándares de calidad y eficacia en el flujo de trabajo para casi cualquier aspecto del proceso de producción.
  • El sistema de producción de audio integrado Nuage reúne el conjunto de requisitos que define los nuevos estándares de calidad y eficacia en el flujo de trabajo para casi cualquier aspecto del proceso de producción.
  • Sistema de producción completamente integrado en cada detalle. Una fusión perfecta del software Steinberg con el hardware Yamaha ofrece una velocidad de operación y edición sin precedentes.
  • La interfaz Nuage se compone de unidades Nuage Fader, cada una con 16 canales fader y knob, y unidades Nuage Master que incluyen todos los controles necesarios para la operación de transporte, edición, monitoreo y más.
  • Se pueden conmutar y controlar directamente hasta tres DAWS desde una unidad de fader Nuage (Nuendo, Cubase y Pro Tools, por ejemplo) para que el diálogo, los efectos de sonido, la música de fondo y otros elementos de un gran proyecto puedan ser fáciles y eficientes. reunidos para la mezcla final.
  • El procesamiento de señal de Nuage es manejado por una computadora estándar de propósito general. No se requieren tarjetas DSP dedicadas que limiten las opciones del sistema. El sistema informático se puede configurar para que coincida con los requisitos de carga de trabajo con la máxima eficacia.
  • La infraestructura de audio de Nuage se basa en la red Dante, lo que proporciona una flexibilidad superior del sistema y una gran capacidad de expansión futura. El audio se puede compartir con las consolas de mezclas digitales de la serie CL a través de unidades de interfaz de la serie R equipadas con Dante como Rio3224-D y otros, o el convertidor Dante / MADI de RMio64-D puede usarse para compartir audio con dispositivos de formato MADI externos.

Aspen 7272HD 3G

Broadcast / Kramer

El conmutador de enrutamiento 7272HD-3G Aspen 72 x 72 3G HD-SDI de Sierra Video es un conmutador de enrutamiento de montaje en rack 4RU con 72 entradas y 72 salidas que admite señales de video SDI, HD-SDI, 3G HD-SDI y enlace dual HD-SDI.

Puede cambiar cualquiera o todas las entradas a cualquiera o todas las salidas. El Aspen 7272HD-3G se puede controlar de forma remota utilizando una interfaz en serie RS-232/485/422 o un puerto Ethernet mediante el uso de las páginas web integradas para la configuración y el funcionamiento.


  • Velocidad máxima de datos 2.97Gbps
  • Operación multi-estándar; SDI (SMPTE 259M y SMPTE 344M), HD-SDI (SMPTE 292M), 3G HD-SDI (SMPTE 424M) y enlace dual HD-SDI (SMPTE 372M)
  • Rango del sistema de 500 '(155 m) a velocidad de datos máxima
  • Sincronización de los modos de reprogramación 5: automático, bypass, 3G HD-SDI, HD-SDI y SD por entrada
  • RS-232, RS-485, RS-422 y control de Ethernet
  • Admite el protocolo TCP / IP con el conector RJ-45 del panel posterior
  • Control de navegador WEB

Z/IPStream R/2

Broadcast / Omnia

Procesador-codificador multi-formato para hasta 8 canales de audio Omnia en un chasis de 1RU.
  • Procesamiento y codificación de flujo de hasta ocho programas de audio en 1 unidad de rack
  • Disponible en modelos con 3 bandas o transformación completa Omnia.9
  • AES/EBU y Livewire/AES67/S de audio
  • Codificar un programa en múltiples tasas de bits para aplicaciones de transmisión de adaptación. De Apple HLS y formatos Smooth Streaming de Microsoft son compatibles AAC-LC, HE-AAC, HE-AAC v2, XHE-AAC y de codificación de flujo de MP3 a partir de 16 kbps a 320 kbps, dependiendo de códec utilizado
  • XHE-AAC para la transmisión a baja tasa de bits
  • Fuentes de alimentación duales y doble puerto Gigabit Ethernet para fiable, 24/7 operación
  • Proceso y codificación se pueden utilizar de forma independiente si se desea
  • Proceso y codificar el mismo programa de audio en múltiples formatos. Al mismo tiempo enviar los flujos codificados a múltiples destinos
  • Plataformas de servidor compatibles incluyen IceCast, SHOUTcast, SHOUTcast v2, Adobe Flash Media Server, Wowza, así como Triton Digital, centro de atención, Akamai y otros servicios de streaming populares
  • Incluye soporte para flujos de multidifusión RTP y RTP
  • Incorporada en el servidor HTTP puede servir directamente a los arroyos HLS
  • HTML5 basado en web de control remoto para la administración
  • Soporte SNMP permite la supervisión directa de su sistema de gestión SNMP, o puede recibir alertas vía e-mail
  • Se integra en el flujo de trabajo: API REST-ful permite un control total de su aplicación para iniciar/detener corrientes, cambiar las fuentes de audio o insertar contenido de audio de archivos; o monitorear varios dispositivos simultáneamente
  • Software dedicado IP remota de control con instrumentos de prueba (RTA, FFT, osciloscopios, el volumen de medición) para los ajustes de procesamiento de audio cuando se utiliza el procesamiento Omnia.9
  • Nueva metadatos flexible permite R/2 para aceptar los metadatos de sistemas múltiples de play-out
